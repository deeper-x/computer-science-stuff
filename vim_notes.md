split window:
```
:sp file_name
```

browse between splitted windows:
```
double <CTRL> and w
```

paste in insert mode:
```
[<ctrl> and <maiusc> and R] +
```

search/replace w/confirmation:
```
:%s/search/replace/gc
```

open new tab:
```
:tabnew file_name
```

browse tabs:
```
<CTRL> and <PAGE_UP>
<CTRL> and <PAGE_DOWN>
```

get current file_name:
```
:echo @%
```
